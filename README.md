## Q&A

* **Why did you start this project?**
	- To see how far I can go with implementing a custom 3D graphics engine in pure Luau (and for it to be able to run smoothly on Roblox)
	- To be able to mimic [hailtododongo](https://www.youtube.com/@hailtododongo)'s custom rendering pipeline for the N64
	- Because I like tsoding (a word coined by Mr. [Zozin](https://www.youtube.com/@Tsoding) means "programming for fun")
